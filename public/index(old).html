About

The goal of Learn Current Events is to improve the general public's awareness of global events through testing users with procedurally generated questions. We use the Machine learning technique of classification to continually improve the quality of the questions as users answer them. We are aware that not every question is perfect, although we are constantly working to improve them. Each question is the title of an article sited in the answer response. We encourage each and every user to visit and read each article. Especially the ones that you answered incorrectly.
